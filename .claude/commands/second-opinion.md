# セカンドオピニオン - Second Opinion

## Step 1: テーマの入力

以下のメッセージを表示してユーザーの入力を待つ:

```
複数の意見を聞きたいテーマを入力してください。（3人の回答者が独立して検討します）
```

---

## Step 2: 独立エージェントのバックグラウンド並列起動

3つのエージェントを **Task toolで `run_in_background: true` を指定して並列起動**。
必ず全てを**同時に1つのメッセージで**呼び出す。

各エージェントへのプロンプト（ラベル A/B/C を変えて）:
```
あなたは「回答者 [ラベル]」です。

以下について、あなた独自の視点で深く検討し、提案してください。
他の回答者を意識せず、あなた自身の最善の分析と提案を行ってください。

【検討テーマ】
（ユーザーが指定した内容）

【出力形式】
## 回答者 [ラベル] の見解

### 要約（3行以内）

### 状況分析
### 提案するアプローチ
### トレードオフ分析
### 懸念点・注意事項

### 結論
- 最終的な推奨
- 確信度: ⭐⭐⭐⭐⭐（5段階）
```

## Step 3: 進捗表示

```
🚀 セカンドオピニオンを起動しました（3名）

📊 実行状況:
  👤 回答者 A ... 検討中
  👤 回答者 B ... 検討中
  👤 回答者 C ... 検討中

⏳ 全回答者の完了を待機しています...
```

## Step 4: 結果の収集

**TaskOutput tool** で各エージェントの結果を収集（`block: true`）
