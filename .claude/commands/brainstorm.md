# ブレインストーミング - Brainstorm

## Step 1: テーマの入力

以下のメッセージを表示してユーザーの入力を待つ:

```
アイデアを出したいテーマを入力してください。（5チームが並列で約15個のアイデアを収集します）
```

---

## Step 2: アイデア生成エージェントのバックグラウンド並列起動

5個のエージェントを **Task toolで `run_in_background: true` を指定して並列起動**。
必ず5つを**同時に1つのメッセージで**呼び出す。

| エージェント | 切り口 |
|------------|-------|
| 1 | 既存の改善・最適化 |
| 2 | 全く新しいアプローチ |
| 3 | ユーザー視点・UX重視 |
| 4 | 技術的チャレンジ |
| 5 | シンプル・ミニマル |

各エージェントへのプロンプト:
```
以下のテーマについて、「[切り口]」の観点から素早くアイデアを出してください。

【テーマ】
（ユーザーが指定した内容）

【担当アイデア数】
約 3 個

【ルール】
- 1アイデア = 1〜2文で簡潔に
- 実現可能性は一旦無視
- 批判せず、量を出す

【出力形式】
1. **アイデア名**: 説明（1文）
2. **アイデア名**: 説明（1文）
```

## Step 3: 進捗表示

```
🚀 ブレインストーミングを起動しました

📊 実行状況:
  🔧 改善・最適化チーム    ... アイデア出し中
  🚀 革新・新規チーム      ... アイデア出し中
  👤 UX・ユーザー体験チーム ... アイデア出し中
  ⚙️ 技術チャレンジチーム  ... アイデア出し中
  ✨ シンプル・即効チーム  ... アイデア出し中

⏳ 全チームの完了を待機しています...
```

## Step 4: 結果の収集

**TaskOutput tool** で各エージェントの結果を収集（`block: true`）
