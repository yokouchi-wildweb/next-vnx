# Sample Game Data

開発用サンプルゲームのデータ。

## ディレクトリ構造

```
game/
├── assets/
│   ├── backgrounds/   # 背景画像
│   ├── characters/    # キャラクター立ち絵
│   └── audio/         # BGM, SE, ボイス
├── scenes/            # シーン定義
└── config.json        # ゲーム設定（予定）
```

## 用途

- エンジン開発時の動作確認用データ
- 将来的にはDBから取得するデータの「モック」として機能
